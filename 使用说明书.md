# ESW 使用说明书

## 一、软件简介

ESW 是一款面向英语教学与备考的**词汇分析工具**，主要用于分析历年真题（高考英语或其他英语考试）中的单词分布情况。它可以帮助教师和学生：

- 快速查询某个单词在历年真题中出现的频率和语境
- 按年份查看词汇分布变化趋势
- 将单词自动归类（课标词 / 真题词 / 拓展词 / 超纲词 / 专有词）
- 对一段文本（如试卷、文章）进行批量词汇分析，快速掌握词汇构成

ESW 基于课标词库（3100 词）和导入的历年真题语料进行智能分类，让词汇复习更具针对性。

## 三、真题导入功能

### 如何导入

1. 点击左侧面板顶部的 **「真题导入」** 按钮
2. 在弹出的菜单中选择导入方式：
   - **导入文件夹**：选择一个文件夹，软件会递归扫描其中所有 .txt / .doc / .docx 文件
   - **导入文件**：手动选择单个或多个真题文件
   - **上次使用的文件夹/文件**：快速重新导入上次的路径（仅在上次导入路径有效时显示）

### 支持的文件格式

| 格式 | 说明 |
|------|------|
| `.txt` | 纯文本格式，UTF-8 编码 |
| `.doc` | 旧版 Word 文档 |
| `.docx` | 新版 Word 文档（自动提取纯文本内容） |

### 文件命名建议

建议真题文件以 **年份开头** 命名（例如 `2024年高考英语全国卷I.txt`），这样软件可以自动提取年份用于词频年份分布统计。

### 导入后能看到什么

- 左侧面板的文件列表会列出所有导入的语料文件
- 每个文件默认**处于选中状态**（蓝色高亮），你可以点击取消不需要参与分析的文件
- 底部统计栏实时显示当前选中的文件数量
- **软件启动时会自动重新导入上次使用的真题路径**，无需每次手动导入

### 多选与管理

- **单选/多选**：点击文件即可切换选中/取消状态，选中的文件才会参与单词检测
- **右键菜单**：在文件上右键可弹出菜单，支持：
  - **打开**：用系统默认程序打开文件
  - **编辑**：用记事本打开编辑文件
  - **删除**：永久删除文件（会二次确认）

## 四、单词检测功能

### 如何输入单词检测

1. 在右侧面板顶部的搜索框中输入要查询的英文单词
2. 按 **Enter 键** 或点击 **「开始检测」** 按钮
3. 软件会在所有**已选中的语料文件**中搜索该单词

### 检测结果包含什么信息

检测完成后，右侧面板会展示：

- **词频总数**：该单词在所选语料中出现的总次数
- **分类标签**：自动判断该词属于五个分类中的哪一个
- **词性与词义**（仅课标词）：显示该词在课标词库中的词性和中文释义
- **词频年份分布**：以卡片形式展示各年份的出现次数
- **上下文匹配列表**：每个匹配项包含：
  - 序号
  - 关键词前后的上下文（左侧前文 + 蓝色关键词 + 右侧后文，后文逐词循环着色）
  - 出处文件名

### 交互操作

- **点击关键词**：打开原文文件并自动定位到关键词位置，支持在原文中选中文本后编辑替换
- **点击年份卡片**：筛选出该年份的匹配结果
- **点击"取消年份筛选"**：恢复显示全部结果
- **鼠标悬停关键词**：弹出扇形菜单，可快速将单词标记为"拓展词"、"超纲词"或"专有词"

## 六、五个分类标签的含义

| 分类 | 颜色 | 含义 |
|------|------|------|
| **课标词** | 蓝色 | 该单词存在于课标词库（3100 词）中，有词性和释义信息 |
| **真题词** | 橙色 | 该词在导入的真题语料中出现过，但不在课标词库中（也未手动标记） |
| **拓展词** | 绿色 | 用户手动添加的词汇，关联到某个课标词作为其"变形/派生/同义词" |
| **超纲词** | 红色 | 在真题语料中出现但不在课标词库中，由用户手动标记为"超纲" |
| **专有词** | 灰色 | 不在课标词库中 → 通常是日常阅读中遇到的专有词汇，包括人名地名等 |

### 分类优先级

单词检测时，分类按以下优先级自动判定：

1. 用户手动标记的**超纲词**（最高优先级）
2. 用户手动添加的**拓展词**
3. 课标词库中的**课标词**
4. 反向索引自动匹配的**拓展词**（课标词 JSON 中预定义的拓展形式）
5. 用户手动标记的**专有词**
6. 在真题语料中出现 → **真题词**
7. 不在真题语料中 → **专有词**

## 七、拓展词添加与管理

### 什么是拓展词

拓展词是指课标词的变形、派生词或同义词。例如 `happily` 可作为 `happy` 的拓展词。添加拓展词后，检测到该词时会自动归类为"拓展词"并关联到对应的课标词。

### 如何添加拓展词

**方法一：从检测结果添加**

1. 检测一个真题词（分类显示为"真题词"）
2. 在左侧面板检测信息区，点击 **「+ 拓展词」** 按钮
3. 在弹出的对话框中输入或选择对应的**基础课标词**
4. 点击确认

**方法二：从上下文结果添加**

鼠标悬停在结果列表中的蓝色关键词上，会弹出扇形菜单，点击 **「拓展」** 按钮。

**方法三：从导入文本分析添加**

在导入文本分析模式下，点击单词旁的 **「筛选」** 按钮，在弹出的对话框中选择"拓展词"。

### 管理拓展词

点击右侧面板的 **「拓展词汇」** 统计卡片，弹出管理对话框：

- **查看列表**：按添加顺序（最新在前）展示所有拓展词及对应的基础课标词
- **点击单词**：直接跳转检测该词
- **编辑**：修改拓展词关联的基础课标词
- **删除**：移除拓展词标记
- **导出为 Excel**：将拓展词导出到 `3100课标词_拓展.xlsx`，在课标词模板的"拓展"列中显示对应拓展词

## 八、超纲词与专有词的手动标记

### 标记超纲词

当检测到某个词在真题语料中出现但不在课标词库中时，可以将其标记为"超纲词"：

- 在检测结果中，点击左侧面板检测信息区，该词为真题词时会显示操作入口
- 或在上下文结果中鼠标悬停关键词，弹出扇形菜单后点击 **「超纲」**
- 标记后该词归类变为红色"超纲词"标签

### 标记专有词

对于不在课标词库中的单词（如专有词汇、人名地名等），可以标记为"专有词"：

- 在上下文结果中鼠标悬停关键词，弹出扇形菜单后点击 **「题外」**
- 在导入文本分析模式中，可通过"筛选→跳过"来批量标记专有词

### 管理已标记词汇

点击右侧面板的 **「超纲词汇」** 或 **「专有词汇」** 统计卡片，可以：

- 查看所有已标记的词汇列表
- 点击单词直接检测
- 删除标记（恢复为自动分类）
- 导出为 Excel 文件

## 九、导入文本分析模式

### 什么是导入文本分析模式

当你需要分析**一整段英文文本**（如试卷、文章、作文）中的词汇构成时，可以使用此模式。软件会自动提取文本中的所有英文单词并逐一分类。

### 如何进入

**方法一：粘贴文本**

在搜索框中直接粘贴包含多个英文单词的文本（至少 2 个单词且含空格），软件会自动检测并切换到导入文本分析模式。

**方法二：导入文档**

点击工具栏的 **「文档导入」** 按钮，选择一个 .txt 或 .docx 文件，文件内容会自动填入搜索框并开始分析。

### 分析面板说明

- **返回 / 清除按钮**：退出导入文本分析模式
- **筛选标签**：按五个分类筛选单词列表，每个标签显示该分类下的单词数量
- **单词列表**：每行显示一个单词，包含：
  - 分类颜色条
  - 单词本身
  - 分类标签
  - 在当前文本中的出现频次
  - 操作按钮（筛选 / 移出 / 跳转）
- **点击单词**：跳转到该单词的详细检测结果

### 批量筛选功能

在导入文本分析模式中，点击未归类真题词的 **「筛选」** 按钮，会弹出分类对话框：

- 输入课标词 → 添加为**拓展词**

分类后自动跳转到下一个未归类的真题词，实现**连续批量归类**。

### 从检测结果返回分析

在导入文本分析中点击某个单词查看详细检测结果后，顶部会出现 **「返回段落分析」** 按钮，点击可回到之前的分析面板，且新添加的拓展词/超纲词标记会自动同步。

## 十二、常见问题与提示

### Q1：为什么导入文件夹后没有显示文件？

确保文件夹中包含 .txt、.doc 或 .docx 格式的文件。导入文件夹模式会**递归扫描子文件夹**，导入文件模式则**不递归**。

### Q2：为什么检测某个单词没有结果？

请检查：
- 左侧文件列表中是否有**选中**的真题文件（蓝色高亮）
- 该单词在选中文件的原文中是否确实出现
- 注意软件使用完整单词匹配（whole word），不会匹配单词的一部分

### Q3：年份分布没有数据？

年份是根据文件名开头提取的（如 `2024年真题.txt` → 2024 年）。请确保文件名以四位年份数字开头（2000-2099）。

### Q4：如何导出拓展词列表？

点击"拓展词汇"统计卡片，在弹出对话框中点击右上角的**导出图标**按钮，会生成 `3100课标词_拓展.xlsx` 文件。需要本地已安装 Python 和 openpyxl 库。

### Q5：超纲词和专有词有什么区别？

- **超纲词**：在真题语料中出现了，但不在课标词库中 → 说明是考试中出现过的"超纲"词汇
- **专有词**：不在课标词库中 → 通常是日常阅读中遇到的专有词汇，包括人名地名等

### Q6：软件配置保存在哪里？

所有配置（导入路径、历史记录、拓展词、手动标记等）保存在 `D:\UserData\Desktop\eva\esw_config.json`，删除该文件会重置所有配置。

### Q7：如何编辑真题文件中的单词？

在检测结果列表中点击蓝色关键词，会打开原文查看窗口并自动定位到该词的位置。在窗口中**选中文本**后点击「编辑选中」按钮，可以将选中内容替换为其他单词。

### Q8：导入的 .docx 文件乱码怎么办？

软件会从 .docx 中提取纯文本，不支持复杂排版、表格或图片中的文字。如果提取不完整，建议先将文档另存为 .txt 格式再导入。
